# Timer

### 概要
- 残り時間をカウントするアプリです。
- 筋トレのレストタイムを計るのに使用してます。

### Versions
Swift | Xcode
-|-
Swift 5.0 | Xcode12.4

### 使用フレームワーク
- SwiftUI
- UIKit
- AVFoundation

### 機能
- 設定した時間を過ぎると音楽が流れる
- 任意の秒数設定が可能
